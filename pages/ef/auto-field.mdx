## Automatic field

| Legend | Description             |
| ------ | ----------------------- |
| ✏️      | Automatic Managed       |
| ✔️      | Allow to take effect    |
| ❌      | Disallow to take effect |
| ⚠️      | Throw error             |

<br/>

### Automatic Attributes

#### Creation time / Last modification time

**AutoCreationTime** / **AutoLastWriteTime**

|  | Add  | Delete | Modify |
| ---- | :----: | :----: | :----: |
| **Auto**      | ✏️    | ✔️  | ✏️  |
| **Preserve**  | ✏️  | ✔️  | ❌   |
| **Free** | ✔️    | ✔️    | ✔️    |

<br/>

**RowLock**

When the field value is **not null**, the value of the specified column is prohibited from changing.

The **database value** is **null**:

|              | Add  | Delete | Modify | Modify columns |
| ------------ | :--: | :----: | :----: | :------------: |
| **Auto**     |  ✔️   |   ✔️    |   ✔️    |       ✔️        |
| **Preserve** |  ✔️   |   ✔️    |   ✔️    |       ✔️        |
| **Free**     |  ✔️   |   ✔️    |   ✔️    |       ✔️        |

The **database value** is **not null**:

|              | Add  | Delete | Modify | Modify columns |
| ------------ | :--: | :----: | :----: | :------------: |
| **Auto**     |  ✔️   |   ⚠️    |   ⚠️    |       ⚠️        |
| **Preserve** |  ✔️   |   ⚠️    |   ❌    |       ❌        |
| **Free**     |  ✔️   |   ✔️    |   ✔️    |       ✔️        |
