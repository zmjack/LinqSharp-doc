## Automatic field

| Legend | Description             |
| ------ | ----------------------- |
| ✏️      | Automatic Managed       |
| ✔️      | Allow to take effect    |
| ❌      | Disallow to take effect |
| ⚠️      | Throw error             |

<br/>

| ---- | ------ | ------ | ------ |
| **Auto**      | ✏️    | ✏️    | ✔️    |
| **Preserve**  | ✔️    | ❌    | ✔️    |
| **Specified** | ✔️    | ✔️    | ✔️    |
